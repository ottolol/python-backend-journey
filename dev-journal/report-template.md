Каждый понедельник (или в начале новой недели) мы будем отвечать всего на три вопроса:

1. Что я уже умею?
2. Что вызывает страх или непонимание?
3. Что я хочу научиться делать к концу недели?

Пример коммитов:
feat: add playground with basic Python variables
docs: add first development journal
chore: initialize project structure
refactor: rename variables in playground

Префиксы - это называется Conventional Commits:
feat: — новая функциональность
fix: — исправление ошибки
docs: — документация
refactor: — изменение кода без изменения поведения
test: — тесты
chore: — служебные изменения (структура проекта, настройки и т.д.)
